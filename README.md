# Scrapy_DoubanTop250_Books
通过Scrapy框架爬取豆瓣图书TOP250（Scrapy/re/SimpleCookie）
## 豆瓣图书 Top250 详情爬虫
本项目是一个基于 Scrapy 框架的爬虫，用于抓取豆瓣图书 Top250 的详细信息。爬虫首先访问 Top250 列表页，提取每本书的详情页链接和基本信息（书名、评分、评价人数等），然后进入详情页，提取作者、出版社、出版日期、页数、定价、ISBN 等详细字段。为了应对豆瓣的反爬机制，代码中加入了从浏览器获取的 Cookie 模拟登录状态。

### 功能特点
Cookie 模拟登录：通过解析浏览器 Cookie 字符串，在请求中携带登录态，降低被反爬的风险。

分页抓取：自动识别并跟进列表页的“下一页”链接，实现全量抓取。

字段丰富：从详情页提取包括书名、作者、出版社、出版日期、页数、定价、ISBN 在内的 10 个字段。

数据清洗：对文本中的空白字符进行统一处理（如合并换行、空格），确保数据整洁。

结构清晰：使用 Scrapy 的 Item 定义数据结构，便于后续存储（如导出 CSV/JSON 或存入数据库）。

### 技术栈
Python 3.8+

Scrapy 2.x

正则表达式（re）

http.cookies（用于解析 Cookie 字符串）
